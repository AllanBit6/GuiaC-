Condiciones

Las estructuras de condicion o condicionales en C++, son indispensables a la hora de aprender a a programar en C++ y otros tantos lenguajes de programacion. Nos permiten definir ciertas condiciones logicas dentro de nuestro algoritmo, estos nos ayudan a determinar ciertas acciones o "caminos" que se ejecutaran de ser la condicion cumplida. Por ejemplo, si pudieramos registrar a un usuario dentro de nuestro programa, pero que solo le deje crear una cuenta si el usuario es mayor de 18 años, nos seria de gran ayuda una condicion, que nos permitira verificar si se cumple tal requisito, y si no, llevar a cabo una serie de instrucciones. Estas nos permiten preveer distintos escenarios dentro de la ejecucion de nuestra aplicacion segun creamos de interes.

Existen varios tipos de condicionales, los cuales tienen diferente funcionalidad y proposito, que nos serviran para distintas ocaciones o escenarios que puedan darse dentro de nuestro algoritmo. Ya cuando los aplicamos, dependera de nosotros saber cual de todas ellas usar para la efecectividad del mismo dentro de nuestro algoritmo. Sin mas que decir, te presento...A las condicionales!

  • Condicion If
  • Condicion Else If
  • Condicion Else

Comenzaremos a explicar cada una (y me refiero a cada detalle, y todo lo relacionado jaja) de las estructuras, y como es su sintaxis en C++, al final de la seccion, dejare algunos videos para que puedas aprender con mayor exactitud de que se hablara dentro del curso. Pero antes, quiero enseñarte algo que son los operadores relacionales y logicos. Los cuales son:

  • Igual: ==
  • Diferente: !=
  • Mayor: >
  • Menor: !=
  • Mayor igual: >=
  • Menor igual: <=
  • Or: &&
  • And: ||
  • Not: !;

Condicional If

Comenzaremos con la condiconal If la cual tambien se conoce como estructura selectiva de casos simples, la cual nos posibilita realizar un conjunto de acciones de ser verdadera la condicion que queremos, por ejemplo, que un numero sea mayor a 10, que una variable sea igual a la letra 'a', etc. En otras palabras...nos permite modificar el flujo del programa a nuestro antojo segun se cumplan ciertas condicionaes. A continuacion te presentamos la sintaxis de la condicional if.

Linea 1: Aqui va la palabra reservada if y dentro de parentesis, la condicion que queremos verificar, y luego abrimos llaves.

Linea 2 a 5: Aqui va todo lo que se ejecutara si se cumple la condicion.

Linea 3: Cerramos llaves de la condicion

Cabe resaltar, que todo esto va dentro de la funcion principal int main(), con su respectivo return 0; para cerrar el programa.

No importa si no entendiste nada de lo que acabas de ver, ya que ahora toca ejemplificar, y con esto, seguro entenderas :)

	 
	 if (condicion){ //Por ejemplo num <= 1
	 
	 cout <<  "Se cumple la condicion :D" << endl;
		
	 }
	 
	 //Resto del codigo

Ejemplo 1 

Deseamos un programa en el que el usuario debe ingresar por teclado, la respuesta a la suma de 2 numeros, en caso de que se ingrese el resultado correcto, se mostrara un mensaje por consola felicitandolo.

Veamos, primero vamos a crear el esqueleto de nuestro programa, que consiste en las librerias, la funcion principal y return 0. Bien entonces, vamos a declarar una variable que almacene la suma de 30 y 50, que son enteros, y como son enteros, entonces crearemos una variable int resultado; que sea entero para almacenarlo. Dentro de la funcion principal, mandamos un mensaje por consola para pedir al usuario que ingrese la respuesta, y por medio de un cin leer la variable resultado;

Una vez que el usuario haya ingresado la cantidad, creamos nuestra condicion if, y la condicion (la que esta entre parentesis) es que: cuando resultado sea igual igual a la suma de 30 + 20, que haga lo que hay dentro de las llaves de la condicion. Lo cual es que, por medio de un cout mandar un mensajito de felicitaciones. Y si no se cumple, entonces se omite y termina el programa. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
	int resultado;
	cout <<  "Cuanto es 30 + 20?" << endl;
	cin >> resultado;
	 
  	if (resultado == 30 + 20){
	 
	   cout <<  "Felicitaciones :D" << endl;
	   cout <<  "Respuesta correcta!" << endl;
		
	 }
	 
  return 0;
  }

Condicion Else 

Seguiremos con la condicional Else la cual tambien se conoce como estructura selectiva de casos dobles, ya que nos permite contenmplar que acciones tomar, cuando dada la condicion del if sea falsa, o sea, prevee las dos posibilidades de la condición (Se cumple, o no se cumple), en otras palabras, nos deja modificar el flujo del programa segun se cumpla o no una condicion, tomando acciones en ambos casos. . A continuacion te presentamos la sintaxis de la condicional else if.

Linea 1 a 5:Aqui ira todo el bloque de if que acabamos de ver hace rato.

Linea 6 a 9: Empezamos poniendo la palabra reservada else, y seguido, abrimos las llaves para el bloque de codigo que se ejecuta si no se cumple la condicion, la cual, por cierto, ya no se escribe

Linea 10: Cerramos llaves de la condicion else.

Una cuestion que siempre hay que tomar en cuenta, es que para usar un else es necesario ponerlo justamente donde terminan las llaves del bloque if ya que de lo contrario, no funcionara, porque no hay una condicion anterior que verificar.

Veamos un ejemplo!

	 
	 if (condicion) { //Por ejemplo num <= 1
	 
	 cout <<  "Se cumple la condicion :D" << endl;
		
	 }
	 else { //Si no se cumple
	 
	 cout <<  "No se cumple la condicion :(" << endl;
		
	 }
	 
	 //Resto del codigo

El diagrama de flujo de Condicional Else seria el siguiente:

Ejemplo 1 

Retomamos el ejemplo anterior. Deseamos un programa en el que el usuario debe ingresar por teclado, la respuesta a la suma de 2 numeros, en caso de que se ingrese el resultado correcto, se mostrara un mensaje por consola felicitandolo. Pero si no se cumple, se muestre un mensaje diciendo que es una respuesta incorrecta.

Como en el ejemplo anterior, dejamos todo tal y como estaba, solo que ahora, agregaremos un bloque de codigo mas luego del if, el cual sera nuestro caso por si la condicion no se cumple, unicamente mostrando con un cout dentro de else que la respuesta es incorrecta, y finaliza el programa, pero si se cumpliese, entonces que muestre el mensaje de felicitaciones como lo hicimos anteriormente. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
	int resultado;
	cout <<  "Cuanto es 30 + 20?" << endl;
	cin >> resultado;
	 
  	if (resultado == 30 + 20){
	 
	   cout <<  "Felicitaciones :D" << endl;
	   cout <<  "Respuesta correcta!" << endl;
		
	 }
	 else {
	 
	   cout <<  "Respuesta es incorrecta..." << endl;
		
	 }
	 
  return 0;
  }

Condicion Else If 

Como ultimo, veremos una condicion, es que como que el if y el else tuvieran un hijo, y surgiera el else if, el cual es una estructura selectiva de casos multiples, ya que nos permite poner otras condiciones, por si acaso, la condicion principal sea falsa, entonces procede a comprobar otras series de condiciones, las cuales pueden ser las que queramos. Podria decirse que el else if, es para tener un camino por si la primera cuestion no se cumple, entonces hacer otra cosa, pero no pasar directo al else, sino comprobar otra condicion antes. A continuacion te presentamos la sintaxis de la condicional else if.

Linea 1 a 5: Aqui ira todo el bloque de if que acabamos de ver hace rato.

Linea 6 a 13: Empezamos poniendo las palabras reservadas else if, y seguido, otra condicion, abrimos las llaves para el bloque de codigo que se ejecuta si no se cumple la primera condicion, pero si la segunda.

Linea 14 a 18: Mismo bloque de else que hicimos hace rato.

Una cuestion que siempre hay que tomar en cuenta, es que para usar un else es necesario ponerlo justamente donde terminan las llaves del bloque if ya que de lo contrario, no funcionara, porque no hay una condicion anterior que verificar.

Veamos un ejemplo!

	 
	 if (condicion) { //Por ejemplo num <= 2
	 
	 cout << "Se cumple la condicion :D" << endl;
		
	 }
	 else if (otra_condicion) { 
	 /* No se cumple la  1ra pero si la 2da 
	 por ejemplo num > 2*/
	 
	 cout << "Se cumplio la 2da condicion :o" << endl;
		
	 }
	 else { //Si no se cumple
	 
	 cout << "No se cumple ninguna condicion :(" << endl;
		
	 }
	 
	 //Resto del codigo

Ejemplo 1 

Retomamos el ejemplo anterior. Deseamos un programa en el que el usuario debe ingresar por teclado, la respuesta a la suma de 2 numeros, en caso de que se ingrese el resultado correcto, se mostrara un mensaje por consola felicitandolo. Si no es la respuesta, pero se acerco en un rango de 10 "menor a 10 o mayor a 10" se le dira que es incorrecto, pero estuvo cerca. Pero si no se cumple ninguna, se muestre un mensaje diciendo que es una respuesta incorrecta.

Como en el ejemplo anterior, dejamos todo tal y como estaba con los bloques de if y else iguales, pero en medio de ambas, pondremos un else if en el cual, la condicion sera que si es mayor a 40, y menor a 60, que muestre que la respuesta estuvo cerca, pero no es lo correcto. Y asi de simple tenemos 3 situaciones distintas para nuestro programa facilmente. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
	int resultado;
	cout <<  "Cuanto es 30 + 20?" << endl;
	cin >> resultado;
	 
  	if (resultado == 30 + 20){
	 
	   cout <<  "Felicitaciones :D" << endl;
	   cout <<  "Respuesta correcta!" << endl;
		
	 }
	else if (resultado > 40 && resultado < 60){
	 
	   cout <<  "Cerca!" << endl;
	   cout <<  "Pero es incorrecto :(" << endl;
		
	 }
	 else {
	 
	   cout <<  "Respuesta es incorrecta..." << endl;
		
	 }
	 
  return 0;
  }
