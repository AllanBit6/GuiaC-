Estructuras

de datos

En la seccion pasada estuvimos estudiando los arregos y matrices, pero...sabias que podemos tener algo como una estructura de matrices? y esta vez podemos almacenar arreglos y matrices de varios tipos! (int, char, string) y declararlo para varias ocasiones, por ejemplo, para almacenar datos de una persona, de un libro, producto o algo que requiera ser declarado varias veces.

Las estructuras de datos nos permiten almacenar varios tipos de arreglos para usarse en algo que podriamos llamar "Variables de estructura" los cuales poseeran los mismos atributos de la estructura, y ahora, vamos a estudiar eso.

Struct

Bueno, las Estructuras de datos se pueden declarar tanto fuera de la funcion principal, como dentro. Pero igual se hacen de la misma manera. Y vamos a ver como es su sintaxis en C++.

Linea 1:Ponemos la palabra reservada struct y seguido, le asignamos un nombre a nuestra estructura, y abrimos parentesis.

Linea 2 a 7: En esta parte, declararemos tooodas las variables que creamos necesarias para nuestra estructura, en este caso, solo declaramos 3.

Linea 8: Cerramos las llaves, y seguido, podemos declarar nuestras variables de tipo estructura, en este caso, tenemos 2, y al final, cerramos con un punto y coma, como siempre.

	 
	 struct  estructura {
	 
	    char  name [30];
	    int  num;
	    float  cent;
		
	 } variableStruct, variableStruct2;
	 
	 //Resto del codigo

Acceder a un atributo 

Todo bien, pero ahora, como podemos acceder a una variable de nuestra variable estructura? Por si quisieramos imprimirla o leer datos sobre ella? Pues es sencillo, ya que unicamente necesitamos escribir el nombre de nuestra variable estructura, un punto, y luego la variable a la que queremos acceder variableStruct.variable y ahora veremos su sintaxis en C++.

Tanto para leer e imprimir es lo mismo, podemos acceder a cualquier variable que queramos.

	 
	struct  estructura { 
	 
	    char  name [30];
	    int  num;
	    float  cent;
		
	 } variableStruct;

	  /*Acceder a la variable 
	 de nuestra variable Struct*/ 
	 cin >> variableStruct.name;
	 //Imprimir de la misma forma
	 cout << variableStruct.name << endl;

Ejemplo 1 

Escribir un programa que lea algunos atributos de 2 libros, como titulo, autor, unidades y precio, y luego los muestre en pantalla.

Bueno, pues declaramos nuestra funcion principal y todo nuestro esqueleto en general, para despues, declarar una estructura a la que llamaremos libros, y dentro de ella, declaramos unas variables y arreglos, los cuales corresponderan a titulo, autor, unid, y precio , a las cadenas les asignamos un indice de "30" por si acaso, y por ultimo declaramos nuestras dos variables de estructura para 2 libros.

Ya dentro, con un cin.getline leemos las cadenas, y con cin leemos las cantidades, accediendo con un punto a cada una de ellas, y repetimos para el libro 2.

Ya por ultimo, imprimimos todo de la manera tradicional, y con esto ya usamos estructuras para recoger datos de 2 libros. 


 #include <iostream.h>
	 
 using namespace std;
 		   
 int main () {		

	struct libros {
	 
	char  titulo [30];
	char  autor [30];
	int  unid;
	float  precio;
		
	 } libro1, libro2;

	 //Leer los datos

	 cin.getline(libro1.titulo, 30);
	 cin.getline(libro2.titulo, 30);

	 cin.getline(libro1.autor, 30);
	 cin.getline(libro2.autor, 30);

	 cin >>libro1.unid;
	 cin >>libro2.unid;

	 cin >>libro1.precio;
	 cin >>libro2.precio;	

	 //Imprimir los datos

	 cout << libro1.titulo << endl;
	 cout << libro1.autor << endl;
	 cout << libro1.unid << endl;	
	 cout << libro1.precio << endl;
	 
	 cout << libro2.titulo << endl;
	 cout << libro2.autor << endl;
	 cout << libro2.unid << endl;	
	 cout << libro2.precio << endl;


	 return 0;
	}
