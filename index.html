¿Qué es la programación?

La programación es el proceso utilizado para idear y ordenar las acciones necesarias para realizar un proyecto. Se ouede definir como el proceso de darle instrucciones a la computadora para realizar determinada acción o cumplir un objetivo. En la actualidad, la noción de programación se encuentra muy asociada a la creación de aplicaciones de informática y videojuegos. En este sentido, es el proceso por el cual una persona desarrolla un programa, valiéndose de una herramienta que le permita escribir el código (el cual puede estar en uno o varios lenguajes, como C++, Java y Python, entre muchos otros) y de otra que sea capaz de “traducirlo” a lo que se conoce como lenguaje de máquina, que puede "comprender" el microprocesador.

¿Qué es un programa?

Un programa es un conjunto de instrucciones ya compiladas que se ejecutan algoritmicamente, los cuales fueron desarrollados dentro de lenguajes y entornos diversos. En este caso en particular, programas en C++, como puede ser todo el paquete de Adobe, Office o videjuegos con Unreal Engine como Geometry Dash o Clash of Clans. ¿Qué es un Algoritmo?

En el dia a dia, se emplean algoritmos frecuentemente para resolver problemas multiples, desde cambiar un foco o preparar café, hasta definir lo que una alarma activará cuando detecte la presencia no deseada de alguien. Algunos ejemplos son los manuales de usuario, que muestran algoritmos para usar algún dispositivo o producto, o las instrucciones y ordenes que recibe un trabajador del jefe. Algunos ejemplos en matemática son el algoritmo de multiplicación, para calcular el producto, el algoritmo de la división para calcular el cociente de dos números, el algoritmo de Euclides para obtener el máximo común divisor de dos enteros positivos, o el método de Gauss para resolver un sistema de ecuaciones lineales. En términos de programación, un algorpitmo es una secuencia de pasos lógicos que permiten solucionar un problema.

Podemos apreciar en la imagen de la derecha, un Diagrama de Flujo el cual es una forma grafica de representar un algoritmo, de manera rapida y sencilla. Tenemos una serie de pasos para interpretarlo. en el cual apreciamos un problema, el cual es que la lampara no enciende, por lo que procedemos.

*Esta el foco quemado?: Si lo esta, hay que cambiarle el foco, pero si no. Comprar una lampara nueva.

¿Qué es un Lenguaje de Programacion?

Es un lenguaje formal que, mediante una serie de instrucciones, le permite a un programador escribir un conjunto de órdenes, acciones consecutivas, datos y algoritmos para, de esa forma, crear programas que controlen el comportamiento físico y lógico de una máquina.

Mediante este lenguaje se comunican el programador y la máquina, permitiendo especificar, de forma precisa, aspectos varios.

Para explicarlo mejor (en otras y con menos palabras), el lenguaje de programación es un sistema estructurado de comunicación, el cual está conformado por conjuntos de símbolos, palabras claves, reglas semánticas y sintácticas que permiten el entendimiento entre un programador y una máquina.

¿Qué es una variable y una constante?

En programación, una variable está formada por un espacio en el sistema de almacenaje (memoria principal de un ordenador) y un nombre simbólico (un identificador) que está asociado a dicho espacio. Ese espacio contiene una cantidad de información conocida o desconocida, es decir un valor.

El nombre de la variable es la forma usual de referirse al valor almacenado: esta separación entre nombre y contenido permite que el nombre sea usado independientemente de la información exacta que representa. Normalmente son tipos de datos

En la siguiente tabla, podrás ver los tipos de datos a los que las variables pueden corresponder.

  • Int = numeros Enteros (1, 23, 1004, 290, 0)
  • Char = caracteres (a, A, @, /, -)
  • Float = decimales (0.1, 12.4, 8.5)
  • Double = decimales dobles (16.25, 490.60, 0.76)
  • Bool = booleano ( si o no, TRUE o FALSE, 1 o 0)
  • String = cadenas de caracteres (frases o texto, Manzana, Camino, Caballo)

Una constante en C++, es una forma parecida a una variable, pero la diferencia es que las constantes tienen un valor estático que no puede ser modificable durante el programa, sino solo asignable su valor desde que se declara, y puede ser de cualquier tipo, tal y como una variable.

Historia de C++

C++ es un lenguaje de programación diseñado en 1979 por Bjarne Stroustrup. La intención de su creación fue extender al lenguaje de programación C mecanismos que permiten la manipulación de objetos. En ese sentido, desde el punto de vista de los lenguajes orientados a objetos, C++ es un lenguaje híbrido.

El nombre "C++" fue propuesto por Rick Mascitti en el año 1983, cuando el lenguaje fue utilizado por primera vez fuera de un laboratorio científico. Antes se había usado el nombre "C con clases". En C++, la expresión "C++" significa "incremento de C" y se refiere a que C++ es una extensión de C.

Comentarios 

Los comentarios son forma de, como su nombre lo dice, comentar a lo largo del código, para tener una guía de que es lo que se esta haciendo, estos no son influyentes para el código, ya que son omitidos por el compilador.

Hay 2 formas de comentar en C++, una forma para comentar una sola linea, y otra para comentar lineas multiples.

Para una sola linea basta con escribir " //Comentario de una sola linea ". Para una multiples lines se usa " /*Comentario de multiples lineas*/ ". Puedes ver los ejemplos en el bloque de codigo de la derecha. 

		
	 // Comentario de una linea
	 
	 /*
	 
	 Comentario de muchas lineas
	 pueden ser las que sean
	 
	 */

Librerias

Veremos lo que son las librerias o cabeceras en C++, las cuales son conjuntos de instrucciones que nos permiten usarlas dentro de nuestro código, hay tanto librerias nativas, como librerias externas.

Para importar librerias nativas, bastara con poner " #include <libreria> ". Algunas librerias nativas que podemos importar, son las primeras 9 que muestro en el código de la derecha.

Para importar librerias externas, como la décima que hasta hasta el fondo del codigo, podemos usar " #include "libreriaExt" ". Podemos notar que para liberias internas, se usa "menor que, mayor que", pero en externas, solo se encierra en comillas simples. 

	
	 //Librerias nativas 
	 
	 #include <iostream.h>
	 #include <stdio.h>
	 #include <stdlib.h>
	 #include <cstdlib.h>
	 #include <conio.h>
	 #include <ctime.h>
	 #include <time.h>
	 #include <math.h>
	 #include <fstream.h>
	 
	 //Librerias externas 
	 
	 #include "rlutil.h"

Acortadores 

Es esta parte de código, la cual es " using namespace std; ", sirve para acortar algunas instrucciones, ya que a veces, algunas, requiren mucho espacio de codigo para hacerlo.

Veamos el ejemplo. Ya que cada vez que se usa una instruccion de la libreria estándar empezando con "std: :", entonces se declara usando " using namespace std ". 

	
  using namespace  std;
		
  //Sin acortador 
   std::cout << "Hola Mundo";
	 
  //Con acortador 
	 
  using namespace std;
  cout << "Hola Mundo";

A continuacion la podras ver en el codigo.

" Int " le indica a la funcion, que necesita encontrar un valor de tipo entero, y dentro de parentesis, van los argumentos de la funcion, se abren y cierran llaves, y dentro de ellas va todo el cuerpo del codigo para nuestro programa.

" return 0; " le devuelve a la funcion, un valor, ya que es de tipo entera, y con ella, cerramos la funcion principal, o cerramos el programa.
