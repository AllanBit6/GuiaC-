Menus

Estructuras Condicionales

Los menus de casos, o estructuras condicionales, son estructuras que nos permiten definir multiples casos, dependiendo del valor o condicion que debemos darle para elegir que caso ejecutar, incluso, definir que accion ejecutar si ninguna se cumple, puede que sea el tema mas facil de entender, asi que vamos a empezar

Podemos tener desde el caso 0, hasta donde queramos, y el caso default o por defecto, veamos la sintaxis ahora:

Switch

Comenzaremos con la condiconal If la cual tambien se conoce como estructura selectiva de casos simples, la cual nos posibilita realizar un conjunto de acciones de ser verdadera la condicion que queremos, por ejemplo, que un numero sea mayor a 10, que una variable sea igual a la letra 'a', etc. En otras palabras...nos permite modificar el flujo del programa a nuestro antojo segun se cumplan ciertas condicionaes. A continuacion te presentamos la sintaxis de la condicional if.

Linea 1: Aqui va la palabra reservada switch y dentro de parentesis, este recibe la variable a comparar, abrimos llaves.

Linea 2: Aqui ponemos una palabra reservada llamada case que define un posible caso, el cual sera el caso 1, y seguido, ponemos dos puntos " : "

Linea 3: Aqui viene el bloque de codigo a ejecutarse

Linea 4: Ponemos break para frenar el caso, y por ultimo un punto y coma, y asi con los casos 2 y 3.

Linea 20: Aqui ponemos default que sera como un caso normal, y cerramos con su respectivo break.

Linea 22: Cerramos llaves y con esto tenemos nuestra estructura.

Mejor...veamos un ejemplo :D

	 
	 switch (opcion) { //Es la variable a comparar
	 
	 case 1:
	 	//Bloque de instrucciones 1
	 break;
	 
	 case 2:
	 	//Bloque de instrucciones 2
	 break;
	 
	 case 3:
		 //Bloque de instrucciones 3
	 break; 
	 //casos 1, 2 y 3 son los que puede tomar
	 /*los break sirven para frenar
	 cada opcion, de lo contario, se ejecutan
	 todas*/
	 
	 
	 default: //bloque por defecto
	 //En caso que no se ejecute ninguna anterior
	 break;
		
	 }
	 
	 //Resto del codigo

Ejemplo 1 

Podemos hacer un programa, en el que el usuario pueda escoger 3 opciones, y si escoge una opcion que no esta en la lista, que muestre que es una opcion invalida.

Bien, entonces creamos el esqueleto o la base del programa, creamos una variable de tipo enter que llamamos opcion, luego por medio de un cout mostramos al usuario un mensaje para elegir la opcion, y luego por medio de un cin leemos la variable opcion. En el switch comparamos la variable opcion, y definimos 3 casos con sus respectivos break, y el caso por defecto. En los 3 anteriores, solo imprimiremos un mensaje que muestre la opcion, mientras que en el caso por defecto, ponemos un 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
    int opcion;
    cout <<  "Elija una opcion (1, 2 o 3)" << endl;
    cin >> opcion;
	 
    switch (opcion) { 
	 case 1:
	 cout << "Ha elegido opcion 1" << endl;	
	 break;

	 case 2:
	 cout << "Ha elegido opcion 2" << endl;	
	 break;

	 case 3:
         cout << "Ha elegido opcion 3" << endl;	 
	 break; 
	 
	 default:
	 cout << "Opcion Invalida!" << endl;	
	 break;


	 return 0;
	}
	 
  return 0;
  }
