Bucles

contadores y acumuladores

Los Bucles en C++ son rutinas que nos permiten repetir cierto bloque de codigo siempre y cuando se cumpla una condicion, pero...por ejemplo si quisieramos escribir los numeros de 1 al 100, pues...seria muy tedioso escribirlos 1 por 1! ya que con un bucle, podemos automatizar ese proceso, y veremos como, pero...primero te mostrare cuales son:

  • Bucle While
  • Bucle Do-While
  • Bucle For For
  • For Anidados

Bucle While

El bucle While nos permite repetir un bloque de instrucciones cuando se cumpla una condicion, en este caso, podemos decir, que primero valida la condicion, y luego ejecuta y repite el bloque de codigo. Veamos como se estructura.

Linea 1: Aqui va la palabra reservada while y dentro de parentesis, la condicion que queremos verificar, y luego abrimos llaves.

Linea 2 a 5: Aqui va todo lo que se ejecutara si se cumple la condicion y se repetira.

Linea 3: Cerramos llaves del bucle

Como siempre, todo esto va dentro de la funcion principal, recuerdalo :D

	 
	 while (condicion){ //Por ejemplo num <= 1
	 
	 cout <<  "Se cumple la condicion :D y repite" << endl;
		
	 }
	 
	 //Resto del codigo

El diagrama de flujo de Ciclo While seria el siguiente:

Ejemplo 1

Queremos que el usuario ingrese un numero, y que si ese numero es menor a 100, pues repetira hasta que lo ingrese.

Bueno, para empezar, creamos el esqueleto del programa. Definimos una variable para leer el numero que el usuario ingrese, que en este caso sera num, luego por medio de un cout pedimos que ingrese un numero.

Colocamos la palabra reservada while para abrir nuestro bucle, y luego parentesis para la condicion. Nuestra condicion sera que si num es menor a 100, seguira pidiendo un numero hasta que se pueda, usando el mismo bloque de codigo de entrada de datos que usamos hace rato. Cerramos la llaves del bucle, y con esto tenemos.

. 
	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
	int numero;
	cout <<  "Ingrese numero < a 100: " << endl;
	cin >> numero;
	 
  	while (numero < 100){
	 
		cout <<  "Ingrese numero < a 100: " << endl;
		cin >> numero;
		
	 }
	 
  return 0;
  }

Bucle Do-While 

Seguiremos con el bucle Do-While el cual es casi igual al while, por lo que no profundizare mucho, solo podemos notar que ahora, primero se ejecutara el bloque de codigo, y luego verificara si la condicion se cumple, y seguira repitiendo hasta que la condicion rompa el bucle.Veamos como funciona!

Linea 1: Aqui va la palabra reservada do y abrimos llaves para el bloque de codigo

Linea 2 a 5: Aqui va todo lo que se ejecutara si se cumple la condicion y se repetira.

Linea 3: Cerramos llaves del bucle y ponemos while y la condicion

La condicion siempre se va a validar al final, asi que ten en cuenta eso. :)

	 
	do { 
		
	cout <<  "Ejecuta, verifica y repite" << endl;
		   
	} while (condicion)
		
	//Resto del codigo

El diagrama de flujo de Ciclo Do-While seria el siguiente:

Ejemplo 1 

Queremos crear un programa que salude al usuario, y queremos que el usuario pueda decidir cuando parar el programa y repetir.

Tomamos la misma estructura del while anterior y creamos el esqueleto, luego creamos una variable llamada decision que sera de tipo entera, luego abrimos el bucle con la palabra do, abrimos llaves, y dentro, le pedimos al usuario que ingrese si quiere repetir, siendo que 1 sera si, y 0 sera no. Por ultimo, cerramos las llaves y ponemos while en donde luego, en parentesis, pondremos la condicion. la cual sera que, cuando decision no sea igual a 1, entonces se cerrara, y asi acabamos con este ejemplo. 

	
	#include <iostream.h>
		   
	using namespace std;
		   
	int main () {
		   
	  int decision;
		   
	do {
		   
		cout <<  "Hola" << endl;
		cout <<  "Quiere repetir? 1 o 0" << endl;
		cin >> decision;
			  
		}while (decision == 1);
		   
	return 0;
	}

Bucle For

Para finalizar con esta seccion de bucles, vamos a hablar del bucle for el cual es un estructura ciclica que nos permite determinar el control de cada iteracion en el bucle. En otras palabras, nos permite definir cuantas veces se va a repetir el ciclo. Veamos la sintaxis.

Linea 1 Aqui ponemos la palabra reservada for luego la condicion y abrimos llaves.

Linea 2 a 4: Aqui ira todo el bloque de codigo que se ejecutara.

Linea 5: Cerramos las llaves del ciclo.

Te explicare unicamente como funciona la condicion, la primera parte int i = 0 es una variable llamada iterador, la cual nos permitira controlar cuantas veces se repetira el bucle. Luego, tenemos i <= 5 lo cual es la condicion real, que es que hasta que i sea menor o igual a 5 pues hara lo siguiente. i++ significa que por cada vuelta, se va a autoincrementar, en otras palabras, aumentara en 1 por cada vuelta o iteracion.

	 
	 for(int  i = 0; i <= 5; i++ ) { //Por ejemplo num <= 2
	 
	 cout << "Se ejecuta 5 veces" << endl;
		
	 }
	 
	 //Resto del codigo

El diagrama de flujo de Ciclo For seria el siguiente:

Ejemplo 1 

Queremos crear un programa, en donde el usuario ingrese un numero hasta el cual contar 1 por 1, y luego imprimir toda la cuenta en consola.

Creamos primero nuestro esqueleto, y de una, declaramos una variable de tipo entero, que almacenara el numero hasta el que el usuario quiera contar, en este caso, le pusimos num, luego con un cout vamos a mandar un mensaje, y con un cin leeremos la variable "num".

Bien, para el bucle, usamos la palabra reservada for y dentro de parentesis, declaramos nuestra variable iterador, que siempre sera i , y la inicializamos en 0, ahora continuamos, y como queremos que cuente hasta donde el usuario quiera, entonces lo que hacemos es decirle que repita, y se autoincremente con i++ hasta que i sea igual menor a la variable num.
Como sabemos que la variable i es la que esta contando las vueltas o iteraciones, entonces podemos usarla para mostrar directamente la cuenta, con un cout y concatenando i. cerramos el bucle con llaves, y con esto, finalizamos con el return 0. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
	int num;
	cout <<  "hasta donde quiere contar?" << endl;
	cin >> num;
	 
	 for(int  i = 0; i <= num; i++ ) { 
	 
	 cout << i << endl; 
	 } 
	 
  return 0;
  }
