Entrada y Salida

de datos por consola

Muy bien! ahora podemos comenzar a ver temas de programacion como tal. En esta seccion aprenderemos lo que es la Entrada y Salida de datos o informacion por consola. Esto es sencillo, ya que requiere de muy pocos pasos (lo digo en serio jaja) para realizarla, y que si lo comparamos con otros lenguajes como Java, el proceso resulta mucho mas compacto, Aqui aprenderemos a como leer informacion que el usuario nos digite por el teclado, y asignarla una variable, para posterior mostrarla en Consola. Por lo que vamos a comenzar a programar!

Declaracion de Variables

Como ya vimos, una variables es un espacio de memoria reservado dentro de nuestro ordenador, el cual almacena algun tipo de dato o datos. En este caso, te ensenare a como declarar variables en C++ y sus atributos.

Te explicare linea por linea como se estructura una variable.

Linea 1: En la primera declaramos nuestra funcion principal, que explicamos en el tema anterior.

Linea 2: Linea en blanco para orden.

Linea 3:Bien, pues aqui es donde se declara la variable, asignando el tipo (int), el nombre(numero) y opcionalmente se podria declarar el valor del mismo.

Linea 4: Linea en blanco para orden.

Linea 5: Aqui asignamos su valor, con el cual decimos que numero ser a igual a 5 + 5, lo que nos da 10.

Linea 6: Linea en blanco para orden.

Linea 7: Return 0 para finalizar el programa, el cual vimos anteriormente.

Linea 8: Linea con el cierre de las llaves declaradas en linea 1.


	 int main (){
	 
	 	int numero = 10;
	 
	 	numero = 5 + 5;
	 
	 return 0;
	 
	 }

La sintaxis para declarar una variable en C++ es muy sencilla, la cual es la siguiente:

[Tipo de dato] [nombre de la variable] = [valor];

Podemos ver los ejemplos del codigo a la derecha, con los cuales damos ejemplos de como declara diferentes tipos de variables, y como asignarles valor.

En este caso, el primer ejemplo, declaramos una variable de tipo entera llamada numero, con su respectivo valor. Ahora, con el resto de variables, notamos que no tienen valor asignado aun, pero eso se puede hacer a posterior, como podemos ver, ahora basta solo con poner el nombre de la variable que ya declaramos, y le asignamos el valor que deseemos, siempre y cuando, sea del tipo de dato que declaramos la variable. Analiza el codigo con determinacion.

Otra cosa que podemos decir, es que cuando asignamos un valor numerico a una variable como puede ser float, int o double, se ponen sin encapsular, solo asi, el numero como tal.

Para caracteres, se encapsula el valor dentro de comillas simples, como las siguientes ' '.

Para cadenas de caracteres, es casi lo mismo, pero encapsulado en comillas dobles "".

Para los booleanos, igual, solo bastan con poner TRUE o FALSE.

Recordemos que todo esto de declaracion de variables se hace dentro de la funcion principal. Una ultima cosa que debemos tener en cuenta, es que, tambien podemos declara muchas variables a la vez, solo separando los nombres por comas, como en el ultimo trozo de codigo.

Salida de Texto en Pantalla 

Ejemplo 1 

Para poder imprimir alguna salida de texto en C++, se usa la instruccion cout junto con <<, ten en cuenta, que estos dos siempre andan juntos, no les des mucha importancia, solo recuerda lo que dije, cout y << par obtener cout <<.

Ahora veamos un ejemplo:

Como puedes ver, << sirve como concatenador, ya que primero imprime "Hola mundo" y seguido del mismo " ProgramarGod ", <<endl es para crear un salto de linea, lo cual no es obligatorio, pero nos sirve para darle orden a nuestro mensaje. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
  cout <<  "Hola mundo" << " ProgramarGod " << endl;
	 
  return 0;
  }

Ejemplo 2 

Para este segundo ejemplo, creo que ya sabes como crear la estructura de un programa basico, colocando primero que todo las librerias, luego el acortador, declarar la funcion principal y programar dentro de ella.

Ahora que ya sabes como declarar variables, pues declararemos una variable de tipo entero, con un valor asignado el cual sera 2, y una variable de tipo string, el cual tambien tendra valor, tendra dentro de ella la frase "Ejemplo". Luego de esto, podemos poner nuestro cout y concatenar cada uno de nuestras variables facil, solo poniendo << entre ellas para conectarlas. Y mira que hasta incluso podemos poner cadenas de texto "numero" por el medio solo conectandola de forma que ya vimos, y por ultimo nuestro querido salto de linea. endl 

	
  #include <iostream.h>
  #include "String.h"
  
  using namespace std;
  
  int main () {
	 
  //Imprimiremos este numero en consola 
  int num = 2
  
  //Y esta frase tambien
  string frase = "Ejemplo";
  
  //Concatenamos todo lo que queramos
  
  cout << frase << " numero " << num << endl;
	 
  return 0;
  }

Entrada o lectura de datos

Ejemplo 1 

Si antes usabamos cout para imprimir en consola, ahora usaremos el comando o instruccion cin para imprimir, y en lugar de usar <<. Asi como pasaba con el "cout<<" ahora usaremos >> los juntamos para obtener la instruccion cin>> quedando de esta forma.

Veamos mas ejemplos:

Analicemos ahora que para leer, ponemos exactamente lo mismo, en este caso, queremos leer la variable nombre, unicamente hay que hacer lo inverso, como se observa la sintaxis a la derecha. E imprimimos exactamente igual como vimos anteriormente cout << nombre;

Se que ya sabes que hace el programa, pero...de todas maneras explicare de nuevo :D

Este es un pequeno sistema de bienvenida, el cual primero pedira un nombre digitado por el usuario, y almacenado en una variable de tipo string. Y luego mostramos un mensaje de bienvenida y concatenamos con el nombre que el usuario ingreso. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
  cout <<  "Hola! este es mi programa en C++<<endl;
  cout <<  "Por favor ingresa tu nombre!"<<endl;
  
  string nombre; //Aqui guardaremos el nombre
  
  cin >> nombre; //Leemos el nombre
  
  cout <<  "Hola!" << nombre << "gracias :D"<< endl;
	 
  return 0;
  }

Ejemplo 1 (mejorado) 

Bueno, teniamos antes un pequeno problema con nuestra lectura de datos, porque al usar espacios en la entrada, el programa falla de una u otra manera, podrias intentarlo ahora mismo si tienes el codigo a la mano. Y es que podemos leer de otra manera! usando una instruccion que no muchos usan, getline la cual tiene parentesis, en donde tenemos 2 parametros que llenar, de esta manera (cin, nombre) que podemos notar, son la instruccion de entrada, y la variable a leer. Queadaria getline (cin, variable); Veamos como mejora nuestro codigo con esto.

No es algo tan relevante, pero mejorara de gran manera la funcionalidad de nuestro programa. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
  cout <<  "Hola! este es mi programa en C++<<endl;
  cout <<  "Por favor ingresa tu nombre!"<<endl;
  
  string nombre; //Aqui guardaremos el nombre
  
  getline (cin, nombre);
  
  cout <<  "Hola!" << nombre << "gracias :D"<< endl;
	 
  return 0;
  }

Ejemplo 2 

Aqui tenemos un ejemplo sumamente similar, pero solo que con numeros, aca no seria tan necesario poner un getline, declaramos 2 variables de tipo float o decimal de un digito, y con ellos operamos en la salida a consola, notamos que es super sencillo. 

	
  #include <iostream.h>
	 
  using namespace std;
	 
  int main () {
	 
  float  num1, num2;
  
  
  cout <<  "Ingrese el primer numero" <<endl;
  
  cin >> num1;
  
  cout <<  "Ingrese el segundo numero" <<endl;
  
  cin >> num2;
  

  
  cout << num1 <<"+"<<num2<<"="<<num1 + num2<<endl;
	 
  return 0;
  }
